  4. Press Enter
  5. Type make install and press Enter

This works regardless of where you extracted the folder.

Why Terminal? macOS blocks downloaded files from running when you double-click them (Gatekeeper security). Running from Terminal avoids this completely ‚Äî no workarounds needed.

üì± Mobile (Android / iOS) ‚ñ∂

The mobile app connects to a server running on your computer.

  1. Start the server on your computer using the steps above
  2. Find your computer's IP address:
    macOS In Terminal: ipconfig getifaddr en0
    Windows In cmd: ipconfig ‚Üí look for IPv4 Address
  3. Open the app on your phone
  4. Go to the Settings tab
  5. Enter your server URL, e.g. http://192.168.1.100:5000
  6. Make sure phone and computer are on the same WiFi

Building the mobile app:

make mobile

You'll need Android Studio or Xcode to compile the final app.

üîë YouTube API Key ‚ñ∂

The app needs a Google API key to search YouTube. The installer asks for it during setup, but you can also set it manually:

  1. Go to Google Cloud Console
  2. Create a project and enable YouTube Data API v3
  3. Create an API key
  4. Add it to a .env file in the project folder:
GOOGLE_API_KEY=your_key_here
